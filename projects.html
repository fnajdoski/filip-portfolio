Here are some of my favourite software projects. Each one challenged me to learn something new and deliver a functional solution.

Classic → Lightning Migration

I spearheaded the migration of the Milros Salesforce org from Classic to Lightning. The project involved auditing existing customisations, rebuilding UI components with Lightning Web Components, and training end users. We achieved a smoother workflow and a modern look and feel without disrupting business operations.

Salesforce, Apex, Lightning Web Components, User Training

Commission Calculation App

For a Swiss telecom client, I developed a flexible commission engine that calculates payouts based on dynamic rules. The solution integrates Apex triggers, flows, and Visualforce pages to handle approvals, PDF generation, and complex logic. It reduced manual errors and saved countless hours of processing time.

Apex, Flows, Visualforce, Integration

Angular Recipe App

This academic project is a recipe management platform built with Angular. It allows users to browse, add, edit, and delete recipes. Features include sorting, filtering, detailed ingredient lists, and a statistics dashboard. The API is built with a Node backend and accessed via a dedicated service.
